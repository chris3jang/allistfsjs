## Getting Started
Download the repository above, or clone it using:
```
git clone https://github.com/chris3jang/allistfs
```
In the application's root directory, run:
```
npm install
```
to install dependencies.

## Usage
In the application's root directory, run:
```
npm run start
```
and navigate to 
```
localhost:8080
```
in your web browser.
